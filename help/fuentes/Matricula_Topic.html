APP ACADEMIA: MATRÍCULA


Introducción a Matrícula.


La sección de matricula es la encargada de dar de alta en la academia a los diferentes alumnos que quieran matricularse en esta. El formulario consta de dos partes claves, el apartado Datos del alumno y Datos de matrícula.


Estos dos apartados los veremos más detallados en los puntos que encontrará a continuación. Una de las cosas a tener en cuenta de este formulario es el uso de un temporizador, que tras 90 segundos, desencadenará una alerta que cerrará y borrará los datos introducidos por el usuario por cuestiones de seguridad.



Ilustración 1: Menú Principal Matrícula


Datos del alumno.


La sección Datos del alumno se compone de diferentes campos con restricciones personalizas a los requerimientos del mismo. En la siguiente tabla podrá observar cuales son las diferentes restricciones que tiene aplicada cada campo.


Campo

Restricción

DNI/CIF

El campo DNI solo admite caracteres de tipo numérico, además de calcular automáticamente la letra del DNI gracias a la combinación de dígitos anteriormente introducidos.

Dirección

El campo dirección no dispone de ningún tipo de restricción.

Teléfono

El campo teléfono solo admite caracteres de tipo numérico, además de impedir la introducción de más números a partir de los 9 caracteres introducidos.

Localidad

El campo localidad no cuenta con ningún tipo de restricción.

Provincia de Residencia

El campo residencia curso no cuenta con ningún tipo de restricción.

Curso

El campo curso no cuenta con ningún tipo de restricción.

Necesito documentación

El campo documentación no cuenta con ningún tipo de restricción.

Necesito Certificado

El campo certificado no cuenta con ningún tipo de restricción.

Tipo de matrícula

El campo matrícula no cuenta con ningún tipo de restricción. Este viene preseleccionado para facilitar al usuario el completado del mismo. Según el tipo seleccionado afectará al campo importe.

Fecha

El campo fecha está bloqueado a partir del día anterior al que se está realizando la matrícula.

Importe

El campo importe se encuentra bloqueado. Este calcula automáticamente el precio en función a los diferentes campos anteriormente seleccionados.


Funcionalidad Botones.


Dentro del formulario encontraremos diferentes tipos de botones en los que podremos accionarlos para que ocurra una serie de procesos según el presionado. Las funcionalidades que ofrecen son las siguientes:

→ Limpiar: Limpia todos los campos del cuestionario dejándolo completamente vacío, siempre avisando al usuario de que esta acción será irreversible con una alerta.

→ Cancelar: Cancela la matrícula y vuelve al inicio de la aplicación, siempre avisando al usuario de que esta acción será irreversible con una alerta.

→ Aceptar: Comprueba los datos de la matrícula y si estos son correctos es dada de alta la matrícula en la base de datos, siempre avisando al usuario con una alerta.
